iOS 8 Application Development for iPhone and iPad Using Swift

This iOS Development using Swift training class teaches attendees how to build iOS 8 native applications for iPhone and iPad using Apple's Cocoa Touch framework and the Swift programming language.

Who Should Do ?

Technical Managers, Consultants, Communications Professionals, Software engineers, System Engineers, Network Professionals, Marketing and Sales professional, IT professionals.
